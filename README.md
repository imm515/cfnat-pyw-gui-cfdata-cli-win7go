这是一份关于 **cfnat 本地订阅生成器**（cfnat-pyw-gui）的使用说明文档。该程序旨在通过 GUI 界面或命令行模式，配合 `cfnat` 工具自动筛选优选 IP，并实时生成可供代理客户端使用的本地订阅地址。

---

### **1. 程序简介**

本程序是一个基于 Python 开发的图形化工具，主要功能包括：

* **IP 优选**：调用 `cfnat-windows7-amd64.exe` 扫描并筛选低延迟的 Cloudflare 有效 IP。
* **订阅生成**：根据内置的节点模板（`nodes.txt`），自动将优选出的 IP 替换到节点配置中。
* **本地服务**：内置 HTTP 服务器，提供本地订阅链接（如 `http://127.0.0.1:8888/sub`），方便代理客户端直接添加。
* **防封保护**：具备 IP 频繁切换检测功能，若切换过快（如 10 分钟内连续切换 2 次）将触发警告或强制停止，以防止账号风险。

### **2. 运行环境与准备**

* **操作系统**：支持 Windows 系统（包含针对 Win7 环境的优化）。
* **必要文件**（需放置在同一目录下）：
* `cfnat-windows7-amd64.exe`：核心优选程序。
* `nodes.txt`：节点模板文件，需包含 `vless://` 或 `vmess://` 格式的原始节点信息。
* `locations.json`（可选）：用于显示 IP 所属机场的详细位置信息。



### **3. 使用说明**

#### **GUI 图形界面模式（默认）**

直接运行程序即可打开图形界面：

1. **选择配置**：在下拉菜单中选择预设的批处理（.bat）配置，或选择“自定义配置”。
2. **设置参数**：
* **IP 数**：设置需要扫描的有效 IP 数量（默认 50）。
* **服务端口**：订阅服务器运行的端口（默认 8888）。


3. **启动**：点击“启动”按钮，程序将开始清理残留进程并执行优选任务。
4. **获取订阅**：
* **本地访问**：`http://127.0.0.1:8888/sub`。
* **局域网访问**：使用程序显示的局域网 IP 地址，可供同网络下的手机等设备使用。



#### **CLI 命令行模式**

通过命令行传递参数运行（例如：`python cfnat_sub_gui.pyw --colo HKG --port 8888`）：

* `--colo`：指定机房代码（如 HKG、SGP）。
* `--delay`：延迟阈值，默认 200ms。
* `--sub-only`：仅启动订阅服务器，不进行 IP 优选。

### **4. 核心功能逻辑**

* **自动更新**：当 `cfnat` 筛选出更优的 IP 时，程序会自动更新 `subscription.txt` 文件并更新订阅内容，用户无需在客户端手动刷新。
* **IP 分布统计**：在优选阶段完成后，程序会统计并显示各机场 IP 的分布情况及平均延迟。
* **清理与单例**：启动时会自动检测并关闭残留的 `cfnat` 进程，且程序不支持多开以避免端口冲突。
