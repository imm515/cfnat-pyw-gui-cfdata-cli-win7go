# cfnat 本地订阅生成器 - GUI版本

## 概述

cfnat 本地订阅生成器是一个基于 Cloudflare NAT IP 扫描的本地代理订阅生成工具，提供图形用户界面（GUI），支持智能机房选择、自动延迟配置、扫描统计保存等功能。

## 功能特性

### 核心功能
- ✅ **Cloudflare NAT IP 扫描** - 自动扫描并筛选高质量的 Cloudflare 边缘节点
- ✅ **本地订阅服务** - 启动本地 HTTP 服务器提供订阅链接
- ✅ **图形用户界面** - 简洁直观的 GUI 操作界面
- ✅ **命令行模式** - 同时支持 CLI 模式（无 GUI）

### 智能配置
- ✅ **智能机房选择** - 根据历史扫描数据自动推荐最佳机房
- ✅ **晚高峰检测** - 自动识别晚高峰时段（18:00-22:00）并调整延迟阈值
- ✅ **统计数据持久化** - 保存和加载机房扫描统计数据
- ✅ **机房代码下拉框** - 方便选择常用机房，避免输入错误

### 操作控制
- ✅ **启动/停止/重启** - 完整的扫描生命周期控制
- ✅ **实时日志显示** - 扫描过程实时输出
- ✅ **单实例保护** - 防止重复运行

## 系统要求

- 操作系统：Windows 7 及以上
- Python 版本：Python 3.7+
- 依赖组件：
  - tkinter（Python 标准库，通常已包含）
  - cfnat-windows7-amd64.exe（CFNAT 扫描程序）

## 安装与使用

### 1. 文件准备

确保以下文件在同一目录下：
```
cfnat_sub_gui.pyw       # 主程序（GUI版本）
cfnat-windows7-amd64.exe # CFNAT 扫描程序
nodes.txt                # 节点模板文件
ips-v4.txt               # IP地址列表
```

### 2. 节点模板配置

编辑 `nodes.txt` 文件，配置您的代理节点模板。支持 VLESS 和 VMess 协议。

#### VLESS 模板示例：
```
vless://uuid@example.com:443?encryption=none&security=tls&sni=example.com&type=ws&host=example.com&path=%2Fpath#节点名称
```

#### VMess 模板示例：
```
vmess://base64encoded...
```

### 3. 运行程序

#### GUI 模式（推荐）：
直接双击 `cfnat_sub_gui.pyw` 或在命令行运行：
```bash
python cfnat_sub_gui.pyw
```

#### CLI 模式：
```bash
python cfnat_sub_gui.pyw --colo HKG --delay 200 --task 100
```

### 4. GUI 操作说明

#### 智能配置提示区域
- **时间检测**：显示当前是否为晚高峰（18:00-22:00）
- **建议延迟**：根据时间自动推荐延迟阈值
- **建议机房**：根据历史扫描数据推荐最佳机房
- **上次扫描**：显示上次扫描的低延迟机房统计

#### 扫描配置区域
- **机房代码**：从下拉框选择目标机房（如 HKG、NRT、LAX 等）
- **延迟阈值**：设置 IP 延迟筛选阈值（ms）
- **并发任务**：扫描并发任务数
- **负载数**：每个 IP 测试的端口数
- **提取 IP 数**：从扫描结果中提取的 IP 数量
- **服务端口**：本地订阅服务的 HTTP 端口

#### 操作按钮
- **启动扫描**：开始扫描并生成订阅
- **重启扫描**：停止当前扫描并重新开始
- **停止扫描**：停止当前扫描

### 5. 订阅使用

扫描完成后，可通过以下方式获取订阅：
- 本地访问：`http://127.0.0.1:8888/sub`
- 局域网访问：`http://[本地IP]:8888/sub`

## 配置参数

### 默认配置
```python
{
    'colo': 'HKG',        # 默认机房
    'delay': 200,          # 平时延迟阈值(ms)
    'delay_rush': 300,     # 晚高峰延迟阈值(ms)
    'task': 100,           # 并发任务数
    'num': 3,              # 每个IP端口数
    'ipnum': 50,           # 提取IP数
}
```

### 常用机房代码
- HKG - 香港
- NRT - 东京
- LAX - 洛杉矶
- SJC - 圣何塞
- SEA - 西雅图
- IAD - 阿什本
- ... 更多机房代码可在扫描后从下拉框选择

## 数据文件说明

### 自动生成的文件
- `subscription.txt` - 生成的订阅内容
- `location_cache.json` - 城市代码缓存
- `location_stats.json` - 机房扫描统计
- `cfnat_sub.pid` - 进程ID文件（用于单实例保护）
- `cfnat_run.log` - 运行日志

### 用户配置文件
- `nodes.txt` - 节点模板（需要用户手动配置）
- `ips-v4.txt` - IP地址列表

## 隐私风险检查

### ✅ 隐私安全评估

#### 1. 数据收集与处理
- **本地运行**：所有操作在本地完成，不向第三方服务器发送数据
- **扫描目标**：仅扫描 Cloudflare 公开的边缘节点 IP
- **订阅生成**：订阅内容在本地生成，不经过中间服务器

#### 2. 网络连接
- **出站连接**：
  - 仅连接到 Cloudflare IP 进行延迟测试
  - 本地 HTTP 服务器仅监听本地/局域网请求
- **入站连接**：
  - HTTP 服务器默认监听 127.0.0.1，可配置局域网访问
  - 无自动向外发送数据的行为

#### 3. 数据存储
- **存储位置**：所有数据存储在程序运行目录
- **敏感信息**：
  - 节点模板包含您的代理配置，请妥善保管
  - 不收集或存储个人身份信息
  - 扫描统计仅包含机房性能数据，不含个人信息

#### 4. 推荐安全措施
- ⚠️ 不要将 nodes.txt 分享给他人，包含您的代理配置
- ⚠️ 如开启局域网访问，注意防火墙配置
- ⚠️ 定期备份重要配置文件
- ✅ 建议在安全的网络环境中使用
- ✅ 程序代码开源，可自行审查

### 结论
本工具专注于本地功能实现，无恶意数据收集行为，符合隐私保护要求。

## 命令行参数

```
usage: cfnat_sub_gui.pyw [-h] [--colo COLO] [--delay DELAY] [--task TASK]
                          [--num NUM] [--ipnum IPNUM] [--port PORT] [--debug]
                          [--sub-only] [--ips IPS] [--template TEMPLATE]

optional arguments:
  -h, --help            显示帮助信息
  --colo COLO           机房代码 (默认: HKG)
  --delay DELAY         延迟阈值ms (默认: 200)
  --task TASK           并发任务数 (默认: 100)
  --num NUM             每个IP的端口数 (默认: 3)
  --ipnum IPNUM         IP数量 (默认: 50)
  --port PORT           订阅服务端口 (默认: 8888)
  --debug, -d           调试模式，显示详细输出
  --sub-only            仅启动订阅服务
  --ips IPS             手动指定IP列表(逗号分隔)
  --template TEMPLATE   模版文件路径
```

## 故障排除

### 常见问题

**Q: 程序无法启动？**
A: 确保已安装 Python 3.7+，并且所有依赖文件在同一目录。

**Q: 扫描没有结果？**
A: 检查 ips-v4.txt 是否存在，尝试增加延迟阈值或 IP 数量。

**Q: 订阅无法访问？**
A: 检查防火墙设置，确认 8888 端口未被占用。

**Q: 如何重置统计数据？**
A: 删除 location_stats.json 文件即可重置。
