## 订阅测速补充

- 新增“测速当前订阅”按钮，可直接对当前订阅 IP 做测速
- 首次运行但已有历史 `subscription.txt` 时，也可以直接测速历史订阅 IP
- 测速结果会保留在日志区，不会被后续 `IP刷新` 行覆盖
- 测速文件固定为 `10MB`

## 测速原理

- 使用 Python 标准库 `socket + ssl` 直连目标 IP
- 不走系统代理，也不走环境变量代理
- HTTPS 场景下保留目标站点的 `Host/SNI`
- 按时间窗口统计下载速度，结果显示为 `MB/s`

说明：

- 更适合做不同 IP 之间的相对比较
- 结果反映的是当前订阅 IP 直连测速目标时的表现，不是代理链路的最终出口速度
- README 只补公开功能说明，不包含私有节点、私有路径或历史开发资料
